# Task Management Dashboard

## Project Overview

The **Task Management Dashboard** is a React-based web application that allows users to efficiently manage their tasks. The application supports creating, editing, deleting, searching, and filtering tasks, providing a simple and organized interface for task tracking.

---

## Features

* Add new tasks with title, description, status, and due date
* Edit existing tasks
* Delete tasks
* Filter tasks by status (All, Pending, In Progress, Completed)
* Search tasks by title or description
* Pagination for better task navigation
* Persistent storage using **localStorage**
* Responsive UI built with **Tailwind CSS**

---

## Tech Stack

* **React.js** (Functional Components)
* **React Hooks** (`useState`, `useEffect`)
* **Vite** (Project setup)
* **Tailwind CSS** (Styling)
* **JavaScript (ES6+)**
* **Git & GitHub** (Version control)
* **Netlify** (Deployment)

## Installation & Setup

1. Clone the repository

```
git clone <repository-url>
```

2. Navigate to project folder

```
cd task-management-dashboard
```

3. Install dependencies

```
npm install
```

4. Run development server

```
npm run dev
```

5. Build for production

```
npm run build
```

---

## Deployment

The project is deployed using **Netlify**. After building the project, the `dist` folder is used as the publish directory.
